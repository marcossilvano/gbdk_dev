CREATION
-----------------------------------
Create venv:
python -m venv .venv

Freeze requirements:
pip freeze > requirements.txt

Install requirements:
pip install -r requirements.txt

ACTIVATION
-----------------------------------
Activate venv:
.\.venv\Scripts\Activate.ps1

Deactivate venv:
deactivate


PACKAGES
-----------------------------------
Install packages (after trying to run and get package error):
pip install requests

Check installed packs:
pip list
